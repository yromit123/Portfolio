Front-end

he front end of a website is the part of the website that users see and interact with. It is responsible for the look and feel of the website, as well as its functionality. The front end is typically made up of three main languages: HTML, CSS, and JavaScript.

Back-end

The backend of a website is the part of the website that users do not see. It is responsible for processing requests from the front end, storing data, and generating responses. The backend is typically made up of one or more programming languages, such as:python,java,node.js,mongoDB,fireBase,Django etc.
